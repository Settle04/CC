## 项目

### 86hk-get — 商品数据自动抓取
基于 Playwright + Claude Code 的电商数据采集工具。自动登录、遍历页面、提取结构化数据并保存为 JSON。
- 技术栈：Node.js、Playwright、Claude Code MCP

### plan-video — Remotion 视频渲染
基于 React 的编程式视频生成方案，用代码控制每一帧。
- 技术栈：React、Remotion、TypeScript
