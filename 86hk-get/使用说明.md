# 86hk.vip 商品数据抓取 — 使用说明

## 环境要求

- Node.js >= 18
- 系统已安装 Google Chrome
- 安装依赖：`npm install playwright`

## 初次登录

在 Claude Code 中完成一次登录（只需一次）：

1. 确保 `.mcp.json` 已配置 Playwright（项目自带）
2. 打开 Claude Code，运行：`打开 https://www.86hk.vip/admin/dashboard`
3. 在弹出的 Chrome 窗口中手动输入账号密码并登录
4. 登录后 Cookie 会自动保存到 `.browser-profile/` 目录

## 日常抓取

```bash
node scraper.js
```

脚本会：
1. 使用已保存的登录状态启动浏览器（系统 Chrome，无头模式）
2. 自动抓取所有在售商品列表和详情（约 1-2 分钟）
3. 保存到 `数据/YYYY-MM-DD/` 目录
4. 自动与上次抓取对比，输出新增/下架/内容变化的商品

## 输出文件

每次抓取在 `数据/YYYY-MM-DD/` 目录下生成：

| 文件 | 说明 |
|------|------|
| `商品列表.json` | 商品列表（名称、佣金、标签、结算规则） |
| `商品详情.json` | 商品详情（14 个板块的内容） |
| `商品完整数据.json` | 列表 + 详情合并 |
| `变化报告.json` | 与上次的差异（有变化时生成） |

## 常见问题

**Q: 抓取报错"登录已过期"？**
A: 在 Claude Code 中打开后台页面，重新手动登录一次即可。

**Q: 想手动触发登录刷新？**
A: 在 Claude Code 中运行：`打开 https://www.86hk.vip/admin/dashboard`，然后登录。

**Q: 只想更新列表不抓详情？**
A: 修改 `scraper.js`，删掉详情抓取循环那一段即可。
